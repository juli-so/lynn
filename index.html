Lynn

A powerful Chrome extension offering tag support for bookmarks, making it easy to manage bookmarks.

Feature

  • Find, open & modify bookmarks easily with tags
  • Sublime-like fuzzy search
  • Nearly all actions can be performed in bulk on multiple bookmarks
  • Vim-like shortcuts
  • Group bookmarks into sessions to open altogether
  • Advanced feature like Markdown reference output & selection multi-add bookmarks
